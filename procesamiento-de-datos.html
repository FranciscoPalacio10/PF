6.1 Comprensión de los datos

6.1.1 Obtención de datos

Para iniciar con este proceso se realiza la conexión a la base de datos utilizando el driver ODBC y con el método que se muestra a continuación:

con <- odbcDriverConnect("Driver=SQLSERVER;Server="";Port="";Database="";Uid="";Pwd="" ")

Una vez obtenida la conexión se procede a obtener la vista generada en la base de datos y se asigna a una variable llamada “DfDatosPrestamos” que es del tipo Data Frame:

DfDatosPrestamos <- data.frame(sqlQuery(con,"SELECT *  FROM [dbo].[View_BigData]"))

   isNa<-is.na(valor)

  if(class(valor) == "character" && !is.na(valor) && (valor == "false" || valor == "true"))

  }

Para evaluar estos métodos de imputación se eligen tres variables que se usaran para determinar la eficiencia del proceso.

  1. codigo_postal el cual tiene un 1.2820513 % de valores faltantes.
  2. cantidad_bancos_ult_24_meses el cual tiene un 22.3076923 % de valores faltantes.
  3. cantidad_bancos el cual tiene un 11.7948718 % de valores faltantes.
