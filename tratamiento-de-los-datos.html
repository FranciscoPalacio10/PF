7 TRATAMIENTO DE LOS DATOS

  1. Para iniciar con el tratamiento de los datos, se importan los datos imputados en el apartado anterior.

  2. Se define la variable a predecir por el modelo de red neuronal, está califica a un préstamo como Bueno(0) y Malo (1).

  • Variable a predecir: CalificacionPrestamo la cual posee:

  • Cantidad de préstamos Buenos (0) - Cantidad de préstamos Malos (1): 125, 398

  • Tipo de dato: Factor

  1. Se eliminan campos con fuerte correlación y en donde los valores son todos 0.
DfPrestamos1 <- DfPrestamos1  %>% select(-X,-fechanacimiento,-es_pensionado,-score_f,-dias_atraso,-compromiso_mensual,-cantidad_pagos_ultimos_4_meses,                              -score_tendencia_ult_6_meses,-es_pnc   ,-bancarizado_ult_24_meses,-cantidad_cuotas_pagadas,-cantidad_cuotas_vencidas,-es_monotributista,-x,-score_tendencia_ult_6_meses,-paid,-outstanding,-over_due,-fecha_creacion,-nse,-es_moroso,-es_inscripto_monotributo,-RatingPrestamo,-Duracion,-es_empleado_domestico,-proceso_legal)
  1. Se observa también que existe cierta variabilidad y diferencias de escala entre los valores mínimos y máximos de los registros medidos. En este sentido, la aplicación de las redes neuronales artificiales exige que las variables de entrada estén todas en el mismo rango, por lo que se hace uso de la función scale para cumplir con esta condición:
preProcessParams <- preProcess(DfPrestamos1[,c(1:67)], method = c("scale"))

DfPrestamos1[, c(1:67)] <- predict(preProcessParams,DfPrestamos1[, c(1:67)] )

DfPrestamosNormalizado <- DfPrestamos1

Una vez obtenido el conjuto de datos escalado se lo asigna a una variable que se usa en el resto del proyecto llamada DfPrestamosNormalizado.

De la cual tenemos:

  • Total de columnas : 68

  • Total de filas : 523

  • Total de variables numéricas : 68.

  • Total de variables carácter: 0.

Exportar data frame

write.csv(DfPrestamosNormalizado,'DfPrestamosNormalizado.csv')
